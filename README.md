# Kotlin Calculator 📱

Uma calculadora simples, moderna e responsiva desenvolvida para Android utilizando **Kotlin** e **Jetpack Compose**.

---

### ✨ Recursos
- **Operações Básicas**: Adição, subtração, multiplicação e divisão.
- **Funções Extras**: Porcentagem (`%`), inversão de sinal (`±`), limpar (`AC`) e apagar dígito (`⌫`).
- **Interface Premium**: Grid dinâmico de botões com suporte nativo ao Modo Escuro/Claro do sistema.
- **Interação Háptica**: Vibração tátil sutil ao pressionar qualquer tecla.
- **Testes Unitários**: Cobertura de testes automatizados para todas as regras lógicas de cálculo.

### 🛠️ Tecnologias
- Kotlin
- Jetpack Compose (Material Design 3)
- JUnit 4

### 🚀 Como Executar

1. Clone o repositório
```bash
git clone https://github.com/Luan-zanardo/kotlin-calculator.git
```
2. Abra o projeto no Android Studio e execute em seu dispositivo/emulador.
